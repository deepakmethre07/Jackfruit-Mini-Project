# Karnataka Bus Search Application üöç

This is a Python GUI application built using **wxPython** that allows users to search and filter Karnataka bus routes using a CSV dataset.

## Features
- Search buses by source and destination
- Filter by AC / Sleeper buses
- Sort by fare, timing, rating, and duration
- View bus details in a popup window
- Cheapest and fastest bus options
- Uses real CSV-based data

## Files in this Project
- `BusSearchApp.py` ‚Üí Main Python GUI application
- `karnataka_bus_10cities.csv` ‚Üí Bus route dataset

- Python 3.8+
- wxPython
